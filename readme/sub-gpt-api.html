註1: 本功能需綁定付款資訊

註2: GPT-3.5 Turbo無需月費，只需付API流量費

註3: GPT-4以上的模型僅供月費帳戶使用，且仍需要付API流量費

  1. 至 OpenAI官網 點擊右上角的Log in，然後選註冊建立一個OpenAI帳戶，已有帳戶者可跳至下一步
  2. 登入後前往 Billing settings 儲值API用的餘額(Add to credit balance)
  3. 前往 API keys 建立一個API KEY(Create new secret key)，此功能需要手機驗證，建立完畢後按下Copy鍵將API KEY複製起來


  4. 使用快捷鍵【Ctrl+V】將API KEY貼到【API設定】>【ChatGPT設定】的API KEY欄位裡
  5. 貼上後點擊下方的Model選項，如果API KEY正確就會列出所有可用模型，請選擇你要使用的模型(目前無訂閱的用戶(Free)只能使用GPT-3.5 Turbo)， 設定完畢後請記得儲存您的設定
  6. 不同帳戶可使用的AI模型也不同，大致如下(表格日期：2024/04/08)
    Free (無須月費) Plus Team, Enterprise
    GPT-3.5 GPT-3.5, GPT-4 GPT-3.5, GPT-4, DALL·E, Browsing, Advanced Data Analysis, and more
  7. 儲存完畢後請回到【翻譯設定】將【翻譯引擎】更改為【ChatGPT】即可開始使用，下方的選項為無法使用官方API的替代方法，沒有問題就不用設定
  8. 若您的位置無法使用OpenAI API請點此查看解決方法
