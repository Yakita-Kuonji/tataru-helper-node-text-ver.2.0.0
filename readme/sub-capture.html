  • 使用方法

    點選 按鈕或是按下Ctrl+F11開啟【翻譯螢幕文字】視窗，然後在擷取視窗中選取欲翻譯的文字(如下方影片所示)

  • 全螢幕擷取

    使用Google Vision模式時可以點選【全螢幕擷取】按鈕取得全螢幕的文字

  • 擷取模式

    • Google Vision(推薦)

      文字辨識能力最精準，需取得Google憑證才能使用( 查看憑證取得教學 )

    • Tesseract OCR

      不需憑證也能使用，但辨識精準度較Google Vision差，錯字出現率較高，需手動編輯修正

  • 擷取設定

    • 換行切割

      勾選後會以換行符號切割擷取的文字，然後各別翻譯之，欲翻譯對話選項請勾選此選項，若想要翻譯整個文章請取消勾選

    • 編輯擷取文字

      勾選後選取文字不會立即翻譯，而是跳出一個編輯視窗編輯擷取到的文字，可用來手動修正【Tesseract OCR】模式的錯字
