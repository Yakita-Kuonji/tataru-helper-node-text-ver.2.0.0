  • 開啟【重新翻譯 & 自訂翻譯】視窗

    點選Tataru Helepr Node的翻譯文字

  • 重新翻譯

    1. 選擇【翻譯引擎】和【語言】
    2. 點選【重新翻譯】按鈕

  • 新增自訂翻譯

    1. 在【替換前(原文)】輸入欲替換的原文
    2. 在【替換後(自訂翻譯)】輸入替換的文字
    3. 點選【類別】選單，選擇自訂翻譯的類型
    4. 點選【儲存自訂翻譯】按鈕

  • 刪除自訂翻譯

    1. 在【替換前(原文)】輸入欲刪除的自訂翻譯原文
    2. 點選【類別】選單，選擇自訂翻譯的類別
    3. 點選【刪除自訂翻譯】按鈕
