注意事項 1: Google Vision每個月有1000次的免費使用次數，超過後每1000次收費1次( 點此查看定價 )

注意事項 2: 本功能需綁定付款資訊

  1. 前往 Google Cloud Platform 啟用Google Cloud Platform服務，資訊可以任意填寫，本功能需使用Google帳號，若沒有Google帳號請先建立一個
  2. 點此 選擇專案，若無專案請建立一個
  3. 點此 設定你的付款設定
  4. 點此 啟用Google Vision
  5. 前往 這裡 ，點選「建立憑證」建立一個服務帳號(名字隨意)，建立後進入該服務帳號選擇「密鑰」，選擇「添加密鑰」>「JSON」，然後按下創建即可
  6. 開啟Tataru Helper Node設定，點選【API設定】>【設定Google憑證】，開啟上一步下載的JSON檔案即可使用Google Vision擷取螢幕文字，儲存的憑證會永久保存，不必再重新設定
  7. 由於憑證含有敏感資訊，因此憑證檔案請妥善保管，若想移除憑證請至 這裡 移除服務帳號即可，移除後需重新建立一個服務帳號才能使用Google Vision服務
