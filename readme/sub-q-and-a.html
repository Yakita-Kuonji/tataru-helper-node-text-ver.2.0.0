回首頁

  1. 這程式安全嗎

    此程式不會修改FF14的遊戲檔案，翻譯功能使用線上API的方式實現，不會在您的電腦上安裝額外軟體，請放心使用

  2. 這程式怎麼運作的

    此程式會擷取記憶體中的對話字幕，然後利用線上API提交翻譯，最後將翻譯結果顯示在程式中，若有勾選名詞校正，在提交翻譯前會對原文做一些更動讓FFXIV的專有名詞(人名、地名等)得以正確翻譯

  3. 名詞沒有被正確翻譯

    此程式只會修正 對照表 內有的詞彙，若詞彙不在對照表裡將不會被程式修正，只能等對照表更新或是以【回報翻譯錯誤】功能來回報

  4. 這程式支援什麼語言

    支援日文和英文，但翻譯以日文版為主

  5. 能與漢化一起使用嗎

    可以，程式有略過漢化字幕的功能(Beta)

  6. 如果遊戲是日文，聊天頻道是英文，該如何設定

    至【設定】>【翻譯】>將【遊戲語言】設定成日文，【玩家頻道】設定成英文即可

  7. Node視窗無法維持在最上層

    請至FF14的【系統設定】將【顯示模式】改為【無邊框全螢幕模式(Borderless Windowed)】

  8. 每一句都要手動選取翻譯嗎

    不需要，除了對話選項和使用者介面以外都可以自動翻譯，不能自動翻譯的部分也能使用【翻譯螢幕文字】功能來翻譯

  9. 程式無法自動翻譯

    請依序嘗試以下步驟
    1. 安裝.NET 6.0
    2. 更新作業系統至最新版本
    3. 勾選下圖中的兩個選項(頻道任意，可以在不常用的頻道勾選這兩個選項)

    4. 解壓並執行壓縮檔內的fix.bat(滑鼠右鍵>以系統管理員執行)，並按下任意鍵開始修復，修復完畢後重新啟動電腦即可

  10. 如何翻譯對話選項和介面文字

    詳見【翻譯螢幕文字】說明頁

  11. 如何重新翻譯

    滑鼠點選程式的翻譯文字，再點選彈出視窗裡的【重新翻譯】按鈕即可

  12. 哪個翻譯引擎比較好

    目前推薦【有道翻譯】和【彩雲小譯】，但上述翻譯引擎都會有翻不好的時候，遇到這情況請嘗試使用其他翻譯引擎(點選程式的翻譯文字選擇其他翻譯引擎重新翻譯)

  13. 視窗半透明失效

    請將【NVIDIA控制面板】的【廣域設定】恢復成預設值(感謝avbkopye沒牙提供解法)
    或是使用【全新安裝】的方式重新安裝NVIDIA驅動程式(安裝時選擇自訂即可看到此選項)

  14. 要等很久才會顯示翻譯

    本應用程式是使用線上翻譯引擎， 因此翻譯速度取決於自身網路狀況和翻譯引擎方的伺服器狀況

  15. 出現「翻譯錯誤」

    原因是該翻譯引擎無法連線，或是遇到流量限制，只能等翻譯引擎解除限制，或是暫時使用別的翻譯引擎

  16. 如何移動視窗

    按住左上的【十字箭頭】圖示即可移動視窗

  17. 旁白字幕無法擷取

    請使用最新版本的Tataru Helper Node

  18. 無法套用變更後的設定

    更改設定後需點選右下角的【儲存變更】按鈕才會生效

  19. 按鈕有點擋畫面

    開啟【設定】視窗，將【隱藏按鈕】打勾

  20. 如何重置設定

    開啟【設定】視窗，點選右下角的【恢復預設值】按鈕

  21. 如何查看程式錯誤訊息

    Ctrl+F12

  22. 如何更新Tataru Helper Node

    請至這裡下載最新版本

  23. 如何移除Tataru Helper Node

    至【新增或移除程式】移除Tataru Helper Node

    請至這裡查看
